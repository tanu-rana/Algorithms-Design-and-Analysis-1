# Huffman Coding

## Overview

This folder contains a program that creates Huffman Tree for the given alphabet
with weights.

### Execution Steps

To run and test it out for yourself, follow these steps:

```bash
$ javac *.java
$ java HuffmanCoding huffman.txt
```
